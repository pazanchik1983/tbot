# T-Bot — торговый бот для Т-Инвестиций (ОФЗ)

> 👤 **Не программист?** Открой файл [`ИНСТРУКЦИЯ.md`](ИНСТРУКЦИЯ.md) — там пошагово, с картинками, как получить и запустить программу за 20 минут без единой команды.

Десктопное приложение для алгоритмической торговли облигациями федерального займа (ОФЗ) через API Т-Инвестиций с возможностью смены брокера и самообучаемыми ML-агентами.

## Ключевые возможности

- 📈 **Окно графика** свечей с уровнями покупок/продаж и историей сделок
- 📋 Список торговых инструментов (ОФЗ) с фильтрами
- 🤖 **Самообучаемые агенты**: ансамбль технических индикаторов (RSI/MACD/EMA) + ML-фильтр (LightGBM), который дообучается на новых сделках
- 🔌 **Адаптер брокера**: легко переключаться между Т-Инвестициями и другими (заглушки для Финам/BCS подготовлены)
- ⚙️ Гибкие настройки: смена API-ключа и брокера прямо в UI
- 🧪 Песочница (sandbox) для безопасного тестирования
- 📦 Сборка в `.exe` для Windows 11 (PyInstaller)
- 📱 Готовая архитектура под Android-клиент (REST/WebSocket к ядру)

## Структура проекта

```
tbot/
├── core/          # ядро: события, конфиг, шина, ордера, риск-менеджмент
├── brokers/       # адаптеры брокеров (T-Invest, Finam-stub, BCS-stub)
├── agents/        # стратегии и ML-агенты (ансамбль + LightGBM фильтр)
├── ui/            # PyQt6 интерфейс (главное окно, график, настройки)
├── data/          # кэш свечей, история сделок, БД SQLite
├── models/        # обученные ML-модели (.pkl)
├── logs/          # журналы
└── resources/     # иконки, стили
packaging/         # build-скрипты PyInstaller + Inno Setup для .exe
android_client/    # план/заготовка мобильного клиента
docs/              # документация и архитектурные заметки
```

## Требования

- **Windows 11** (для запуска UI и сборки `.exe`); ядро кроссплатформенно.
- **Python 3.11** рекомендуется (поддерживается 3.9–3.12). На Python 3.13 SDK Т-Инвестиций пока не собирается.
- Учётная запись Т-Инвестиций и API-токен (см. `Настройки` в приложении).

Если `pip install tinkoff-investments` падает (пакет временно в карантине на PyPI), ставьте напрямую из git:

```powershell
pip install "git+https://github.com/RussianInvestments/invest-python.git@main"
```

## Быстрый старт (разработка)

```powershell
# Windows 11, Python 3.11+
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python -m tbot
```

## Сборка .exe для Windows 11

```powershell
pip install -r requirements-build.txt
python packaging/build_exe.py
# результат: dist/TBot/TBot.exe + опционально dist/TBot-Setup.exe
```

Подробнее — см. `packaging/README.md`.

## Безопасность

- API-ключ хранится в зашифрованном виде (`keyring` — Windows Credential Manager)
- По умолчанию приложение запускается в **режиме sandbox**
- Боевой режим включается осознанно в Настройках с дополнительным подтверждением
